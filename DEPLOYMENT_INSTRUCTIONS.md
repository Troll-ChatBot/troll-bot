# Инструкции по деплою Troll Chat

## Подготовка к деплою

1. Убедитесь, что у вас установлен Node.js
2. Установите Vercel CLI:
   ```bash
   npm install -g vercel
   ```

## Настройка API-ключа

1. Получите API-ключ от Groq на https://console.groq.com
2. При деплое в Vercel вы сможете указать переменную окружения `GROQ_API_KEY`

## Локальная разработка

1. Установите зависимости:
   ```bash
   npm install
   ```

2. Запустите локальный сервер:
   ```bash
   npm run dev
   ```
   
   Примечание: для локального запуска также потребуется аутентификация в Vercel и настройка переменной окружения

## Деплой на Vercel

1. Выйдите в корневую директорию проекта
2. Выполните команду:
   ```bash
   vercel
   ```
3. Следуйте инструкциям в терминале
4. Когда система запросит переменные окружения, укажите:
   - `GROQ_API_KEY`: ваш API-ключ от Groq

## Безопасность

- Никогда не коммитьте API-ключи в репозиторий
- Используйте только переменные окружения для хранения чувствительных данных
- Не делитесь своим API-ключом с другими

## GitHub репозиторий

1. Создайте новый репозиторий на GitHub
2. Инициализируйте локальный репозиторий:
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   git remote add origin <URL_вашего_репозитория>
   git push -u origin main
   ```
3. После создания репозитория вы можете импортировать его в Vercel для автоматического деплоя

## Проверка работоспособности

После деплоя проверьте, что:
- Сайт доступен по предоставленному URL
- Чат-интерфейс загружается корректно
- Сообщения отправляются и получают ответы от тролль-бота
- Нет ошибок, связанных с API-ключом
